# PROMATCH Telegram Bot

Bot oficial de assinaturas da PROMATCH.

## Fluxo

Telegram → Plano STARTER → PIX → Asaas → Validação → Grupo privado

## Plano atual

STARTER — R$ 49,90/mês

## Tecnologias

- Node.js
- Telegram Bot API
- Asaas API
- Express
- Axios

## Variáveis de ambiente

TELEGRAM_BOT_TOKEN

ASAAS_API_KEY

TELEGRAM_GROUP_ID

ASAAS_ENV

PORT

## Segurança

Nunca adicionar o arquivo `.env` ao repositório.

Tokens do Telegram e chaves da API do Asaas devem ser configurados diretamente no ambiente de hospedagem.
